# 音乐工具箱 · 完整使用说明

> MUSIC TOOLS · 赞美诗工作站
> 进入后是一个「工具箱」首页，里面有两个工具，点卡片进入，左上角 **← 返回** 回到首页，右上角 **?** 随时打开图文教程。

| 工具 | 用途 |
|---|---|
| 🎵 **歌词编辑器**（LRC） | 给音乐打时间轴、导入/导出 `.lrc` 歌词文件 |
| ♩ **简谱编辑器** | 输入简谱音符、标歌词和弦、导出歌曲 JSON（给 musiclib / 青年页用） |

# 一、简谱编辑器（重点）

版本 v3.1。整首歌由 **段落 → 行 → 格子（seg）** 组成，每个格子里有三样东西：**和弦**（上）、**简谱音符**（中）、**歌词**（下）。你在左下角点一个格子选中它，然后用右边的键盘往里填音符。

## 1. 界面速览

- **顶栏标签**：导入 / ⌨ 填歌词 / ⚑ 检查 / ⇔ 对齐 / 预览 / 工具 / 代码
- **上半区**：预览（所见即所得的简谱）、工具、代码（元数据 + 导出）三选一
- **中间状态栏**：显示当前选中位置、音符总数、当前八度/时值/附点/延长号/房子线、选区、剪贴板
- **左下段落区**：所有段落和格子，点格子进入编辑；顶部有搜索框（⌘F）
- **右下键盘区**：所有音符和符号的输入按钮 + 双行简谱面板

## 2. 快速上手（5 步）

1. **填信息**：点「代码」标签，填歌名、艺人、原调、拍号、BPM（ID = 导出文件名）。
2. **建段落**：左下点「+ 新增段落」，给段落起名（主歌/副歌/Bridge…）。
3. **选格子**：点段落里的一个格子，它会高亮，状态栏显示位置。
4. **填音符**：右边键盘先选 **八度** 和 **音值**，再点数字 **1–7**（或 0 休止）。需要延音、附点、小节线、连音等就点对应按钮。
5. **填词加弦**：直接在格子的歌词栏输入歌词、和弦栏输入和弦（如 `C` `Am` `G7`）。
6. **导出**：回「代码」标签 → 「复制完整 JSON 文件」，存成 `ID.json`。

## 3. 音符输入详解

选好格子后，右边键盘从上到下：

- **八度**：低2 / 低1 / **中** / 高1 / 高2（快捷键 `↑ ↓` 升降八度）。
- **音值**：全 `Q` / ½ `W` / ¼ `E`（四分，默认）/ ⅛ `R`（八分）/ ¹⁄₁₆ `T`。先选音值再点音符。
- **输入模式**：**插入 `I`**（在光标处插入）/ **覆盖 `O`**（替换当前音）。
- **音符**：`1 2 3 4 5 6 7` 七个音（Do–Si），`0` 是休止符。

## 4. 基础符号

| 按钮 | 作用 | 快捷键 |
|---|---|---|
| — 延音 | 音延长一拍（`-`） | `\` |
| · 附点 | 附点音符（时值 ×1.5），开关 | `,` |
| 𝄐 延长号 | 自由延长记号，开关 | `F` |
| ␣ 空格 | 插入占位空格 `sp`（对齐用，不发声） | `Space` |

## 5. 临时记号（升降号）

先选，再点音符，记号就加在那个音上：

- ♯ 升 `K` ／ ♭ 降 `J` ／ ♮ 还原 `H` ／ 清除

## 6. 线条与结构

| 按钮 | 作用 | 快捷键 |
|---|---|---|
| `|` 小节线 | 分小节 | `B` |
| `||` 双小节 | 段落分隔 | `M` |
| `||/` 终止线 | 全曲结束 | |
| `|:` 反复开 · `:|` 反复结 · `|:|` 反复段 | 反复记号 | |
| 拍号记号 | 在谱中插入变拍号（如中途 3/4） | |
| ( ) 连音 | 圆滑线，把选中的一串音连起来，开关 | `[` / `S` |
| ⌒ 双连音 | 连接相邻两个音（`~`） | `N` |
| ⌒² 跨音连 | 跨过中间 1 个音连接（`~2`；`~3` 跨 2 个） | `⇧N` |
| 跨线开 / 跨线结 | 手动指定连线的起止 | `]` / `X` |
| 3连 / 5连 | 三连音 / 五连音 | |

## 7. 房子线（反复的 1./2. 括号）

- **1. 房开** / **2. 房开**：第一、第二房子线开始
- **自定义房**：自己填数字的房子线
- **房结**：房子线结束（`]v`）
- **自定义Tok**：手动插入任意 token（也可输双行 `1/5`）

## 8. 双行简谱（上下两行叠着的音）

右下角「双行简谱可视化输入」面板：
1. **上行**、**下行** 各自点数字填音（`sp` 占位、`-` 延音）。
2. 选时值：4分 / 8分 / 16分，可加附点、延长号。
3. 看「预览 token」（如 `1/5`）确认无误，点 **插入双行**。
（快捷键 `/` 快速跳到这个面板。）

## 9. 格子操作

| 按钮 | 作用 | 快捷键 |
|---|---|---|
| ⌫ 删除 | 删掉选中的音/格 | `Bksp` |
| ↩ 撤销 | 撤销上一步 | `⌘Z` |
| ✕ 清空 | 清空当前格的音符 | |
| ⬡ 复制格 | 复制整格（和弦+音符+歌词） | `Alt+C` |
| ⬡ 粘贴格 | 粘到后面 | `Alt+V` |
| ⬡ 覆盖格 | 覆盖当前格 | `Alt+R` |

## 10. 歌词与和弦

- **单格填**：点格子，在它的歌词栏 / 和弦栏直接打字。
- **批量填歌词**：顶栏「⌨ 填歌词」，把整段歌词一次贴进去，按音符自动分配（省得一个字一个字点）。
- 和弦写常见写法即可：`C` `Am` `G7` `F#m` `C/G`…

## 11. 检查 · 对齐 · 整理

- **⚑ 检查**：核对每行的音符数和歌词数对不对得上，出错会提示。
- **⇔ 对齐**：预览行两端对齐（只影响预览排版，**不改数据**）。
- **工具 → 整理格式**：规范化数据格式。

## 12. 试听与试移调（工具标签）

- **播放当前行 / 播放整首 / 停止**：用合成音色试听，检查音对不对。
- **试移调**：填原调和目标调，点「预览」看移调后的样子（只预览，不改原数据）。

## 13. 元数据与导出（代码标签）

先把信息填全：

| 字段 | 说明 |
|---|---|
| ID | **文件名**，如 `zmzq-shunfu` |
| 歌名 / 艺人 / 副标题 | 显示信息 |
| 原调 | 如 `A` |
| 拍号 / BPM | 如 `4/4` / `72` |
| YouTube / MP3 | 链接或文件名 |
| 封面 cover / 歌词 lrc / 乐谱图 scoreImg | 填文件名即可，默认前缀 `https://cecpadua.github.io/resouces/…` |

导出两种：
- **复制 sections 数组**：只要谱面数据
- **复制完整 JSON 文件**：整首歌完整 JSON（存成 `ID.json` 用这个）

导入：顶栏「导入」，把 `sections` 数组或 `var SECTIONS = […]` 粘进去。

## 15. 简谱 token 速查（`n` 字段里长什么样）

| 写法 | 含义 |
|---|---|
| `1`–`7` | 音符 Do–Si；`0` 休止 |
| `1'` `1''` | 高一/高二八度（撇号）；`1,` `1,,` 低八度（逗号） |
| `1_` | 八分音符（一条下划线）；`1_ ` 连着写表示连拍 |
| `1#` `1b` `1=` | 升 / 降 / 还原 |
| `-` | 延音线（延长一拍） |
| `1.` | 附点 |
| `sp` | 占位空格（对齐用，不发声） |
| `|` `||` `||/` | 小节线 / 双小节 / 终止线 |
| `|:` `:|` `|:|` | 反复开 / 反复结 / 反复段 |
| `( … )` | 圆滑线（连音） |
| `~` `~2` `~3` | 双连音 / 跨 1 音 / 跨 2 音连线 |
| `[v1 … ]v` | 第 1 房子线（`[v2` 第 2） |
| `[ts:4/4]` | 谱中变拍号 |
| `1/5` | 双行简谱（上行 1 / 下行 5） |

# 二、歌词编辑器（LRC 打轴）

给一首歌配上「每句歌词在第几秒出现」的时间轴，导出 `.lrc`。

## 1. 打轴流程

1. **⬆ 上传音乐**：选一个音频文件。
2. **准备歌词**（三选一）：
   - **⬆ 上传歌词**：已有 `.lrc` 文件
   - **📄 上传文本**：粘纯文本歌词（每行一句，不用时间戳）
   - **🔍 查找歌词**：在线找
   - 或 **📄 创建歌词**：直接一行一句打进去（会清空当前歌词）
3. **打轴**：点 ▶ 播放，每唱到一句就点那一句，把当前时间戳打上去。
4. **微调**：
   - **偏移**：整体提前/延后几秒（填秒数 → 应用偏移）
   - **同步滚动**：开/关跟随播放高亮
   - **+ 新增歌词**：在末尾补一句
5. **add info**：填歌手 / 歌名 / 专辑 / 制作，写进 LRC 头。
6. **⬇ 下载歌词** 导出 `.lrc`；**⬇ 下载双语LRC** 导出中英双语版。

## 2. 播放控制

- 进度条可拖动，▶ 播放/暂停，右边有音量滑块，左上显示当前时间 `[00:00.00]`。

# 小贴士

- 两个工具的教程可随时点右上角 **?** 重新看。
- 简谱编辑器边填边看「预览」，最直观；填错用 `⌘Z` 撤销。
- 导出前先点 **⚑ 检查**，确认音符数和歌词数对得上，避免上线后错位。
- 导出的 `ID.json` 放进歌库后，musiclib / 青年页 / 投影 都能直接用。
